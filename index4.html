How To

To set a key/value call

https://api.keyval.org/set/ { KEY } / { VALUE }

If you want to auto-generate key you can set key as -

ex:- https://api.keyval.org/-/{VALUE}

To get a value using key

https://api.keyval.org/get/ { KEY }
